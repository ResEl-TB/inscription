Contacter le ResEl

En passant par IRC

IRC est un protocole de messagerie instantanée, très utilisé par les administrateurs ResEl. Si un administrateur est présent en ce moment, c'est de cette façon que vous aurez une réponse le plus rapidement.

Envoyer un mail

Si aucun administrateur n'est disponible en ce moment, vous pouvez utiliser ce formulaire pour envoyer un mail. Nous en tiendrons compte un peu plus tard. Donnez une brève description de votre problème ou requête dans le corps du texte, et remplissez tous les champs.

Le mail sera envoyé immédiatement à inscription@resel.fr.

Pour les gens logeant à la MaisEl mais n'ayant pas de compte informatique à l'école, lorsque vous vous inscrivez (en le demandant par ce formulaire par exemple), vous entrez dans nos bases de données et un mot de passe temporaire vous est alors communiqué (une fois que votre demande d'inscription a été acceptée). Il vous servira pour inscrire votre machine sur le site d'inscription
